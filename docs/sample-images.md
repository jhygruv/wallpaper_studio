# 샘플 이미지 검증 가이드

아래 조합으로 최소 5개 샘플 이미지를 준비해 수동 검증합니다.

## 샘플 세트

1. 날짜 + GPS 모두 있음 (일반 스마트폰 촬영본)
2. 날짜는 있고 GPS 없음 (위치권한 비활성 촬영)
3. EXIF 날짜 없음 + GPS 있음
4. EXIF 거의 없음 (스크린샷/메신저 저장 이미지)
5. 세로 사진(모바일), 가로 사진(데스크탑) 각각 1장

## 검증 항목

- 날짜 텍스트가 예상대로 표시되는지
- GPS 있을 때 자동 위치가 채워지는지
- GPS 없을 때 수동 위치 입력이 정상 동작하는지
- Desktop 4K / Mobile 프리셋이 비율 깨짐 없이 생성되는지
- 다운로드 파일명이 프리셋 해상도를 반영하는지
